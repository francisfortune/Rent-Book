# RentBook - Rental Business Dashboard

RentBook is a web application that allows users to set up and manage their rental businesses efficiently. It supports inventory management, rental tracking, and dashboard analytics.

## Features

- Multi-step business setup:
  - Business Info
  - Location
  - Rental Model
  - Default Return Duration
  - Inventory Setup
- Dynamic inventory management
- Firebase Firestore integration for storing businesses and inventory
- Responsive design for desktop and mobile
- User-friendly dashboard for managing rentals
- Toast notifications for feedback

## Tech Stack

- HTML5 / CSS3 / JavaScript
- Firebase Firestore (Database)
- Tailwind CSS (Styling)
- Vanilla JS for front-end logic
